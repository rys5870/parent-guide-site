# Parenting Guidance Platform

## 🎯 Project Purpose

This website was created as part of a personal portfolio to showcase design, development, and accessibility skills using modern web technologies.  
It is not intended for public use or code reuse.

The platform is designed to help parents strengthen their relationship with their children through thoughtful content and interactive tools.  
It offers a calm, respectful space to explore emotional development, communication, and identity.

---

## 🧩 Features

- 📚 articles on parenting, emotions, and child development  
- 🎧 Podcasts with certified parenting coaches  
- 🎨 Drawing submission form for personalized interpretation  
- 🔍 Category-based filtering for tailored content  
- 🖼️ Image upload with preview  
- 📱 Responsive design for mobile and desktop  
- ♿️ Accessibility support (ARIA roles, keyboard navigation, semantic HTML)  
- 🌐 SEO-friendly structure with clean URLs and proper heading hierarchy  
- 🧠 RTL support for Hebrew content
